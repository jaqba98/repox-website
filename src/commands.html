Commands


Generate Workspace

To create repox workspace use the command below.

repox generate workspace --name=example

The command is responsible for creating a clean repox workspace, ready for use. Generates all required configuration files for writing, testing and linting code.

Argument Alias Required
--name -n Yes

Regenerate Workspace

To regenerate repox workspace use the command below.

repox regenerate workspace --force

The command is responsible for regenerating the existing repox workspace. Generates all required configuration files from scratch.

Argument Alias Required
--force -F Yes
