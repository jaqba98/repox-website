Getting Started

Welcome to Repox, your go-to solution for managing monorepos with ease. Follow the steps below to quickly set up your environment and start using Repox for efficient project management.

1. Prerequisites

Before you begin, ensure that you have the following prerequisites installed on your system:

- Node.js and npm: Repox is an npm package, and it requires Node.js and npm. You can download and install them from https://nodejs.org/

2. Install Repox Globally

Open your terminal or command prompt and run the following command to install Repox globally:

npm install -g repox

This makes Repox available as a command-line tool across your system.

3. Create a New Workspace

Now, you can create a new workspace within your monorepo using the generate workspace command. Replace "my-workspace" with your desired workspace name:

repox generate workspace --name=my-workspace

This sets up a new workspace structure for your projects.

4. Create a New Project

Once your workspace is set up, create a new project within it using the generate command. Replace "my-project" with your desired project name:

repox generate project --name=my-project

This command initializes a new project within your workspace, providing the necessary structure to get started with your development.

5. Start Developing

With your workspace and projects set up, you're ready to dive into development. Utilize Repox to efficiently manage dependencies, build projects, and maintain a structured monorepo.

For detailed information on Repox commands and configuration options, refer to the official Repox documentation.

Now you're all set to make the most of Repox and streamline your monorepo development workflow!
