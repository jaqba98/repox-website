Generate Project

The generate workspace command in Repox allows you to create a new workspace within your monorepo. Workspaces serve as organizational units for your projects, providing a centralized environment for development. Follow the guide below to use this command effectively.

Command Syntax

repox generate workspace --name=my-workspace

Parameters

--name (or -n): Required. Specifies the name of the workspace to be generated.

Usage Examples

Basic Usage

repox generate workspace --name=my-workspace

This command creates a new workspace named "my-workspace" within your monorepo.

Using Aliases

You can use aliases to make the command more concise:

repox g w --name=my-workspace

Here, g is an alias for generate, and w is an alias for workspace.

Command Options

--name (or -n): Specifies the name of the workspace. This parameter is required.

Additional Information

- Workspaces provide an organized structure for managing projects within your monorepo.

- Ensure that the workspace name is unique within your monorepo.

- Aliases (-n, --name, g, w) provide a shorter and more convenient way to use the command.
