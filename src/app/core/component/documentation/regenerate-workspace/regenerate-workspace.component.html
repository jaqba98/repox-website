Regenerate Workspace

The regenerate workspace command in Repox allows you to regenerate an existing workspace within your monorepo. This command is useful when you need to update the workspace configuration or make changes to its structure. Follow the guide below to use this command effectively.

Command Syntax

repox regenerate workspace --force

Options

- --force: Required. Indicates that the regeneration should proceed, overwriting any existing configuration. Use this option to enforce the regeneration.

Usage Example

Basic Usage

repox regenerate workspace --force

This command regenerates the existing workspace, applying any updates to the configuration or structure. The --force option is required to initiate the regeneration.

Additional Information

- The regeneration process may overwrite existing workspace configuration. Ensure you have backups or version control in place if necessary.

- The --force option is mandatory to confirm your intention to regenerate the workspace.
